---
title: 'Create a token to initialize the SDK'
description: 'Create a token to initialize an instance of the Benji Connect SDK.'
sidebarTitle: "Create Token"
api: 'POST https://api-staging.withbenji.com/verify/token/create'
---

<Note>The Bearer token to be used on this API is your **partner access token**</Note>

<ParamField body="user_external_id" type="string" required>
  A unique identifier associated with the user connecting their loyalty reward program
</ParamField>
<ParamField body="display_name" type="string">
  The name of your store you wish to be displayed in the Benji Connect flow. If this is not passed, the name configured on the Benji platform will be used
</ParamField>
<ParamField body="user_ip" type="string">
  The IP of the user currently connecting. Note that this is required if there are any geo specific conditions applied to the partner campaigns.
</ParamField>
<ParamField body="partnership_id" type="string">
  The unique identifier of the partnership on the Benji Platform.
</ParamField>

### Return values 
<ResponseField name="token" type="string" required>
  The created token token to use when initializing the Benji Connect SDK
</ResponseField>

<ResponseExample>
```json 200
{
    "code": "ok",
    "data": {
        "token": "0c371a1c-aa81-11ef-b28c-0a58a9feac02"
    }
}
```

```json 401
{
  "error": "unauthorized_partner",
  "message": "Invalid key, or key is not authorized to make this call"
}
```

```json 500
{
  "error": "unexpected_error",
  "message": "An unexpected error has occured"
}
```
</ResponseExample>
