---
title: 'Create a token to initialize the SDK'
description: 'Create a token to initialize an instance of the Benji Connect SDK or Hosted Page.'
sidebarTitle: "Create Token"
api: 'POST https://api-staging.withbenji.com/verify/token/create'
---

<Note>The Bearer token to be used on this API is your **partner access token**.</Note>

<ParamField body="display_name" type="string" required>
  The partnership display name you wish to be shown in the Benji Connect flow UI _Note: If this is not passed, the name configured on the Benji platform will be used._
</ParamField>
<ParamField body="mode" type="number" required>
  The connection mode for the SDK (Connect = 1, Transfer = 3, and Redeem = 4)
</ParamField>
<ParamField body="partnership_id" type="string">
  The unique identifier of the partnership on the Benji Platform.
</ParamField>
<ParamField body="user_external_id" type="string">
  A unique identifier associated with the user connecting their loyalty reward program, when set flow skips the initial user authorization with the partner and continues to authorize the user with the related merchant/partner.  
</ParamField>
<ParamField body="custom_attributes" type="object" optional>
  Custom attributes for the connection flow
  <Expandable title="properties" defaultOpen="true">
    <ParamField body="actions" type="array" optional>
      Array of actions to be performed during the connection flow
      <Expandable title="action properties" defaultOpen="true">
        <ParamField body="mode" type="number" optional>
          The action mode (Transfer = 3, Redeem = 4)
        </ParamField>
        <ParamField body="amount" type="number" optional>
          The points amount to transfer or redeem from the partner
        </ParamField>
        <ParamField body="allows_editing" type="boolean" optional>
          Determines whether the user can edit the points amount in the UI or not
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

### Example Requests

<AccordionGroup>
  <Accordion title="Minimal Request (Connect mode)">
```json
{
    "mode": 1,
    "display_name": "Reward Cookies",
    "partner_id": 3,
    "partnership_id": 49,
    "merchant_id": 48
}
```
  </Accordion>

  <Accordion title="With user_external_id">
```json
{
    "mode": 1,
    "display_name": "Reward Cookies",
    "partner_id": 3,
    "partnership_id": 49,
    "merchant_id": 48,
    "user_external_id": "1234"
}
```
  </Accordion>

  <Accordion title="With custom_attributes (Transfer mode)">
```json
{
    "mode": 3,
    "display_name": "Reward Cookies",
    "partner_id": 3,
    "partnership_id": 49,
    "merchant_id": 48,
    "custom_attributes": {
        "actions": [
            {
                "mode": 3,
                "amount": 1000,
                "allows_editing": true
            }
        ]
    }
}
```
  </Accordion>
</AccordionGroup>

### Return values 
<ResponseField name="token" type="string" required>
  The created token token to use when initializing the Benji Connect SDK
</ResponseField>

<ResponseExample>
```json 200
{
    "code": "ok",
    "data": {
        "token": "0c371a1c-aa81-11ef-b28c-0a58a9feac02"
    }
}
```

```json 401
{
  "error": "unauthorized_partner",
  "message": "Invalid key, or key is not authorized to make this call"
}
```

```json 500
{
  "error": "unexpected_error",
  "message": "An unexpected error has occured"
}
```
</ResponseExample>
