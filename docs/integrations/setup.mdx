# Integration Setup

Connecting your payment processor to Benji is simple and requires no engineering effort.

## Prerequisites
- Benji account
- Square account

## Steps to Connect Square
1. Go to your Benji Developer Settings.
2. Click the “Connect Square” button.
3. Log in with your Square merchant username and password.
4. Review and accept the requested permissions (we only request the minimum needed).
5. That’s it! Your Square account is now connected to Benji.

If you run into issues, see [Troubleshooting](./troubleshooting.mdx).

## FAQ
**Do I need to write any code?**
> No! Benji handles all the integration work for you.
